RStudio Packages

We use multiple RStudio packages for building web apps, generating reports & documents, and creating web APIs. RStudio provides multiple database connectors and enables interaction with Big Data architecture.

The tidyverse is an opinionated collection of R packages designed for data science. All packages share an underlying philosophy and common APIs.

The ggplot 2 is an enhanced data visualization package for R. Create stunning multi-layered graphics with ease.

The dplyr is the next iteration of plyr, focussing on only data frames. dplyr is faster and has a more consistent API.

The tidymodels framework is a collection of packages for modeling and machine learning using tidyverse principles.

Shiny makes it incredibly easy to build interactive web applications with R. Shiny has automatic “reactive” binding between inputs and outputs and extensive pre-built widgets.

The rmarkdown lets you insert R code into a markdown document. R then generates a final document, in a wide variety of formats, that replaces the R code with its results.

TensorFlow™ is an open-source software library for Machine Intelligence. The R interface to TensorFlow lets you work productively using the high-level Keras and Estimator APIs and the core TensorFlow API.

The tidyr makes it easy to “tidy” your data. Tidy data is data that’s easy to work with: it’s easy to munge (with dplyr), visualise (with ggplot2 or ggvis) and model (with R’s hundreds of modelling packages).
